为什么要重做

老主题陪了我三年，功能堆得越来越多：悬浮预览、打字机一言、自动轮播的即刻条、点赞礼花……每一个单独看都挺有趣，放在一起就成了打断阅读的噪音。这次重做，我给自己定了一条规矩：任何元素，如果它不能让文字更好读，就不配出现在页面上。

好的界面像一张干净的书桌——东西不多，但每一样都放在你伸手就能够到的地方。
大部分文章都是夜里写的，索性把夜色定成了默认。

于是有了现在这个版本：窄栏、暖褐的夜色、一盏琥珀色的灯。导航竖排在左侧，像书脊上的签条；正文两端对齐，字间距放宽，行高两倍，读长文章时眼睛不容易累。日间模式也一并画好了，是一张晒过太阳的暖纸，白天来串门的朋友可以自己拧开台灯。

留下的与拿掉的

点赞留下了。它符合这种私人博客的气质：读者看完后可以轻轻表达喜欢，又不必专门组织一段评论。评论、说说、图集、友链、归档这些核心内容也都在，只是视觉上都退到了文字后面。

备注：链接悬浮预览在 V2 中默认取消。它在桌面端偶尔有用，但会打断阅读、遮挡窄栏内容；移动端也没有 hover。站内链接保留清晰的标题，评论与说说继续提供复制链接。

一点技术备忘

页面切换用的是异步请求，旧页面先"陷下去"，新页面再从下方浮上来，整个过程不到半秒，既保留了单页应用的流畅，又不至于喧宾夺主。核心逻辑大概是这样：

JavaScript
function goto(next) {
  current.classList.add('leaving');      // 旧页陷下去
  setTimeout(() => {
    current.classList.remove('current', 'leaving');
    next.classList.add('current', 'entering');  // 新页浮上来
  }, 280);
}
写这段的时候是凌晨一点四十，电台里在放老歌。

字体没有引入任何外部资源，直接使用系统字族；背景的方格纸纹理也只是两条一像素的渐变线，加起来不到一百字节。整个主题最重的部分，是这些絮絮叨叨的文字本身。
